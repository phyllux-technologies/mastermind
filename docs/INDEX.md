## Quick Start (30 seconds)

1. **Install** — Clone or copy mastermind/ into your workspace.
2. **Setup** — Copy mastermind.md to workspace root. Configure core/projects/_registry.md.
3. **Use** — Add ideas to core/memory/main_memory.md. Run core/processes/improvement_propagation.md.

---

## Core Concepts

- **mastermind.md** — Central control. Read first.
- **core/** — The engine. Memory, processes, projects, meta.
- **Main Memory** — Canonical store for ideas, epiphanies, lessons, insights.
- **Improvement Propagation** — Runs on every new idea; applies improvements across all projects.
- **Mission** — Define in config/mission.md. Aligns improvements.
