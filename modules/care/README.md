# MASTERMIND Care — Care Lens

Adds care/attention scoring to propagation. Ensures improvements don't sacrifice human care. Pirsig: care is prior to technology.

*Attribution: Robert Pirsig, Zen and the Art of Motorcycle Maintenance.*

---

## Install

1. Copy `templates/propagation-questions.md` into Step 2 (Evaluate).
2. Optional: Add Care score (1–5) to Main Memory entries.

---

## Use

When evaluating: **"Are we giving this idea proper attention? Does this preserve care?"**

Avoid superficial improvements. Care is prior.
