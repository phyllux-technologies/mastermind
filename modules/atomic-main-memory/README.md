# MASTERMIND Nucleus (Atomic Main Memory) — MASTERMIND Add-on

Adds Zettelkasten atomicity. One idea per entry. Linked ideas. Fleeting→permanent workflow. Propagation follows links.

*Attribution: Zettelkasten method. Niklas Luhmann. Concepts adapted for MASTERMIND.*

---

## Install

1. Add `links[]` and `state: fleeting|permanent` to Main Memory schema.
2. Copy propagation-questions. Split compound ideas before propagating.

---

## Use

**Atomic:** One idea per entry. Split compound ideas. **Links:** Ideas link to each other. Propagation follows links. **Workflow:** Capture fleeting → process to permanent before propagation.
