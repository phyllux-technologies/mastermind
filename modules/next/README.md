# MASTERMIND Next — GTD Bridge

Propagation produces next actions. Context tags. Weekly review schedule. GTD integration.

*Attribution: David Allen, Getting Things Done.*

---

## Install

1. Add Next Action output template to propagation.
2. Add context tags (computer, phone, etc.) to Main Memory.
3. Schedule weekly review in calendar.

---

## Use

- **Next actions:** Each propagation run → explicit next actions.
- **Contexts:** Tag by @computer, @phone, @errands.
- **Review:** Weekly. Run propagation. Update contexts.
