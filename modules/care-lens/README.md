# MASTERMIND Care (Care Lens) — MASTERMIND Add-on

Adds Pirsig's care/attention dimension to propagation. Ensures improvements don't sacrifice human care. "Are we giving this idea proper attention?"

*Attribution: Robert Pirsig, Zen and the Art of Motorcycle Maintenance.*

---

## Install

1. Copy `templates/propagation-questions.md` into `improvement_propagation.md` Step 2.
2. Optional: Add Care score 1–5 to evaluation.

---

## Use

Care is prior to technology. Before applying an improvement, ask: **"Are we giving this proper attention?"** Reduces superficial improvements. Care scoring: 1 (rushed) to 5 (deep attention).
