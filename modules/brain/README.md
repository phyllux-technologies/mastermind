# MASTERMIND Brain — Second Brain Mode

CODE: Capture, Organize (PARA), Distill, Express. BASB workflow integrated with Main Memory and propagation.

*Attribution: Tiago Forte, Building a Second Brain.*

---

## Install

1. Set up PARA (Projects, Areas, Resources, Archives) in Main Memory or config.
2. Add capture checklist. Add distill prompts.
3. Propagation = Express. Output-focused.

---

## Use

- **Capture:** Easier capture from multiple sources.
- **Organize:** PARA structure.
- **Distill:** "What's the one thing?" Progressive summarization.
- **Express:** Propagate to create output.
