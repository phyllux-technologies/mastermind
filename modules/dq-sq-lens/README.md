# MASTERMIND Pulse (DQ/SQ Lens) — MASTERMIND Add-on

Adds Quantonics/MoQ change vs stasis evaluation. Tag ideas DQ (change), SQ-temporary, SQ-exclusive. Prevents runaway stasis. Better propagation decisions.

*Attribution: Quantonics, Doug Renselle. Pirsig MoQ. Concepts adapted for MASTERMIND.*

---

## Install

1. Copy `templates/propagation-questions.md` content into your `improvement_propagation.md` Step 2 (Evaluate).
2. Add optional `DQ_SQ` column to Main Memory: DQ | SQ-temporary | SQ-exclusive.

---

## Use

When evaluating an idea, ask: **"Does this favor change (DQ) or stasis (SQ)?"**

- **DQ** — Dynamic Quality. Change. Novelty. Moral direction.
- **SQ-temporary** — Static Quality, temporary. Patterns that serve change. Good.
- **SQ-exclusive** — Static Quality, exclusive. Freezing. Avoid.

Complementarity: Both/and over either/or. Include-middle logic. Fewer false dichotomies.
