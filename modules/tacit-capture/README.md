# MASTERMIND Tacit (Tacit Capture) — MASTERMIND Add-on

Adds Polanyi's tacit dimension. "We know more than we can tell." Prompts to surface tacit knowledge. Personal vs shared tagging. Captures experiential wisdom.

*Attribution: Michael Polanyi. The Tacit Dimension. Personal Knowledge.*

---

## Install

1. Add knowledge_type: tacit|explicit, ownership: personal|shared to Main Memory.
2. Copy capture prompts. Use when adding ideas.

---

## Use

**Tacit** — what we can't fully articulate. **Prompts** — surface it. "What do you know that you haven't written down?" **Personal vs shared** — clarify ownership. Capture experiential wisdom before it's lost.
