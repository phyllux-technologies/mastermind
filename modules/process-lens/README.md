# MASTERMIND Duration (Process Lens) — MASTERMIND Add-on

Adds Bergson duration. Ideas have duration, not just date. "How long has this been evolving?" Process over substance. Creative evolution. Novelty over repetition.

*Attribution: Henri Bergson. Creative Evolution. Duration.*

---

## Install

1. Add duration_start, evolving_since to Main Memory. Copy propagation-questions.
2. Novelty vs repetition check in evaluation.

---

## Use

**Duration** — ideas evolve over time. **Process** — focus on improvement process, not fixed outputs. **Creative evolution** — propagation as creative evolution. Encourage genuine novelty. Avoid repetition for its own sake.
