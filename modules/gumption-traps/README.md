# MASTERMIND Gumption (Gumption Traps) — MASTERMIND Add-on

Adds Pirsig's gumption traps to propagation. Out-of-sequence, value rigidity, anxiety. Flag these. Reduces blocked or misdirected improvements. Tacit knowledge capture (knack).

*Attribution: Robert Pirsig, Zen and the Art of Motorcycle Maintenance.*

---

## Install

1. Copy propagation-questions. Add trap checklist.
2. Step 2: trap check. Tag if trap detected.

---

## Use

**Gumption traps:** Out-of-sequence (doing things in wrong order). Value rigidity (stuck in old pattern). Anxiety (fear blocking action). **Knack** — capture tacit experiential wisdom in Main Memory. Flag traps; unblock propagation.
