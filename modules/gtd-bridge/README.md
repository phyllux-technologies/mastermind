# MASTERMIND Next (GTD Bridge) — MASTERMIND Add-on

Adds GTD next actions from propagation. Context tags (computer, phone, etc.). Weekly review schedule. Clear follow-through.

*Attribution: David Allen, Getting Things Done.*

---

## Install

1. Copy propagation-questions. Add context tags to Main Memory.
2. Configure weekly review in config.

---

## Use

Propagation produces **next actions**. Tag by **context**. Run **weekly review** — scheduled propagation. Sync to task system if desired.
